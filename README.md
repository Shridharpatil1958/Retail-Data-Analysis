# Retail Data Analysis Project

## Project Overview
This project performs end-to-end retail data analysis using Python, MySQL, and Power BI to generate business insights and build interactive dashboards.

## Tools Used
- Python (Pandas, NumPy, Matplotlib)
- MySQL
- Power BI

## Project Workflow

1. Data Cleaning using Python
2. Exploratory Data Analysis (EDA)
3. Data Storage in MySQL
4. SQL Business Queries
5. Power BI Dashboard Visualization
